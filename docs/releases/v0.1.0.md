# v0.1.0

## Changelog

- 初始化 TypeScript npm CLI 包结构。
- 新增 `nemoclaw-deploy` 默认命令，等价于 `deploy`。
- 新增 `deploy` 命令，支持 `--config`、`--non-interactive`、`--dry-run`。
- 新增 `doctor` 命令，检查 Node.js、`nemoclaw` 命令和可选 sandbox 状态。
- 新增 `init-config` 命令，可输出或写入 deploy.yaml 模板。
- 新增统一部署上下文、输入校验、YAML 解析和命令执行封装。
- 实现 agent 创建、agent-content 安全上传合并、skills 目录批量安装、advbot channel 配置、network policy 目录批量应用、recover 和验证。
- 增强部署步骤提示，执行时说明每个阶段的目的。
- 增加模板文件：`deploy.yaml`、agent 初始化内容和 advbot network policy。
- 增加 README 使用说明。
